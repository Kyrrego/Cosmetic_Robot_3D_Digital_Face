PROPERTY_EXT_SERIAL_NUMBER 

修改序列号

HDR自动模式的配置

PROPERTY_EXT_AUTO_EXPOSURE_MODE 

深度相机自动曝光模式

PROPERTY_EXT_DEPTH_SCALE 

深度值缩放系数

PROPERTY_EXT_TRIGGER_MODE 

触发模式

PROPERTY_EXT_CONTRAST_MIN 

对比度阈值，此阈值用于删除原始图像中低曝光区域,将删除灰度低于该阈值的区域,阈值越大低曝光区域删除越多.调节范围0~40灰阶,建议设置为5

PROPERTY_EXT_LED_ON_OFF 

是否打开LED灯

PROPERTY_EXT_HDR_BRIGHTNESS 

HDR各级参数

PROPERTY_EXT_TRIGGER_OUT_MODE 

是否开启脉冲输出

PROPERTY_EXT_TRIGGER_IN_MODE 

触发模式

PROPERTY_EXT_LASER_BRIGHTNESS 

激光器亮度

PROPERTY_EXT_LASER_ON_OFF 

激光器开/关

PROPERTY_EXT_SPEED 

拍图速度
