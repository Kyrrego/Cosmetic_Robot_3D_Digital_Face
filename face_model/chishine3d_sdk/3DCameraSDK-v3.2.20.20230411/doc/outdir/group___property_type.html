枚举: 相机的基本属性

枚举值
PROPERTY_GAIN 

增益 调节相机亮度,提高增益会引入噪声,导致深度精度下降。调节范围1~16, 高精度时,建议增益≤3

PROPERTY_EXPOSURE 

曝光值 用于调节相机亮度,数值越大亮度越高,不会引入噪声,但是帧率会随着曝光时间增大而降低,调节范围3000~60000微秒

PROPERTY_FRAMETIME 

帧时间

PROPERTY_FOCUS 

焦距

PROPERTY_ENABLE_AUTO_FOCUS 

是否自动对焦

PROPERTY_ENABLE_AUTO_EXPOSURE 

是否自动曝光

PROPERTY_ENABLE_AUTO_WHITEBALANCE 

是否自动白平衡

PROPERTY_WHITEBALANCE 

白平衡值

PROPERTY_WHITEBALANCE_R 

白平衡R通道

PROPERTY_WHITEBALANCE_B 

白平衡B通道

PROPERTY_WHITEBALANCE_G 

白平衡G通道
