enum HDR_MODE

枚举: 高动态的模式

枚举值
HDR_MODE_OFF 

关闭

HDR_MODE_HIGH_RELECT 

高反模式适用于测高反物体, 会按照设定的曝光级数,增加较低的曝光参数进行多次曝光后进行融合输出。曝光级数由用户设定

HDR_MODE_LOW_RELECT 

暗色模式适用于测深色物体, 会按照设定的曝光级数,增加较高曝光参数进行几次曝光后进行融合输出。曝光级数由用户设定

HDR_MODE_ALL_RELECT 

复合模式适用于测复合表面 会平均分配曝光级数,分别增加较低曝光参数和较高曝光参数来进行多次曝光并融合输出。曝光级数由用户设定

enum AUTO_EXPOSURE_MODE

枚举: 自动曝光模式

枚举值
AUTO_EXPOSURE_MODE_CLOSE 

关闭

AUTO_EXPOSURE_MODE_FIX_FRAMETIME 

固定帧率模式：在当前帧率允许范围下自动实时调节相机的曝光时间和增益, 不调节相机的输出帧率

AUTO_EXPOSURE_MODE_HIGH_QUALITY 

高质量模式：以获得最高质量模型为目标, 自动实时调节相机的曝光时间和增益, 同时会按需调节帧率

AUTO_EXPOSURE_MODE_FORE_GROUND 

近景优先模式：以获得最高质量模型为目标, 自动实时调节相机的曝光时间和增益, 同时会按需调节帧率
