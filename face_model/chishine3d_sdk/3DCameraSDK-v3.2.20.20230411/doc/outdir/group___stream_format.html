深度流和RGB流所支持的所有格式

枚举类型说明

enum STREAM_FORMAT

#include <hpp/Types.hpp>

枚举: 流数据格式

枚举值
STREAM_FORMAT_MJPG 

RGB流的MJPG压缩的数据

STREAM_FORMAT_RGB8 

RGB流的8位红,绿,蓝3通道数据

STREAM_FORMAT_Z16 

深度流的深度图格式, 每一个深度值以unsigned short表示

STREAM_FORMAT_Z16Y8Y8 

深度流的深度图+红外图组合格式, 通过FRAME_DATA_FORMAT_Z16获得深度数据， 通过FRAME_DATA_FORMAT_IR_LEFT获得左红外图, 通过FRAME_DATA_FORMAT_IR_RIGHT获得右红外图

STREAM_FORMAT_PAIR 

深度流的红外图格式，适用于双目相机 通过FRAME_DATA_FORMAT_IR_LEFT获得左红外图, 通过FRAME_DATA_FORMAT_IR_RIGHT获得右红外图

STREAM_FORMAT_H264 

RGB流的H264压缩的数据

STREAM_FORMAT_I8DS 

降采样红外预览格式 通过FRAME_DATA_FORMAT_IR_LEFT获得左红外图, 通过FRAME_DATA_FORMAT_IR_RIGHT获得右红外图， 通过FRAME_DATA_FORMAT_VCENTER_LEFT获得左红外图中光刀中心的y坐标数组, 通过FRAME_DATA_FORMAT_VCENTER_RIGHT获得右红外图中光刀中心的y坐标数组， 通过FRAME_DATA_FORMAT_LASER_WIDTH_LEFT 获得左红外图光刀宽度数组, 通过FRAME_DATA_FORMAT_LASER_WIDTH_RIGHT获得右红外图光刀宽度数组

STREAM_FORMAT_XZ32 

点云输出格式, 通过cs::Pointcloud::generatePointsFromXZ计算成点云

STREAM_FORMAT_GRAY 

深度流的红外图格式，适用于单目相机

STREAM_FORMAT_COUNT 
enum FRAME_DATA_FORMAT

#include <hpp/Types.hpp>

枚举: 帧数据格式，用于获取复合流数据中的指定格式数据起始地址

枚举值
FRAME_DATA_FORMAT_Z16 

深度流的深度图格式, 每一个深度值以unsigned short表示

FRAME_DATA_FORMAT_IR_LEFT 

左红外图数据， 8-bit unsigned char表示一个灰度值

FRAME_DATA_FORMAT_IR_RIGHT 

右红外图数据， unsigned char表示一个灰度值

FRAME_DATA_FORMAT_VCENTER_LEFT 

左红外图光刀中心， unsigned char表示一个y坐标

FRAME_DATA_FORMAT_VCENTER_RIGHT 

右红外图光刀中心， unsigned char表示一个y坐标

FRAME_DATA_FORMAT_LASER_WIDTH_LEFT 

左红外图光刀宽度， unsigned char表示一个宽度值, 光刀太宽时影响点的精确度

FRAME_DATA_FORMAT_LASER_WIDTH_RIGHT 

右红外图光刀宽度， unsigned char表示一个宽度值, 光刀太宽时影响点的精确度
